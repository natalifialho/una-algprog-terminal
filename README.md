Meus Comandos Favoritos:

Aqui estão os comandos que mais utilizei na aula de Terminal:
- `cd`: Usado para navegar entre pastas.
- `dir`: Lista todos os arquivos e pastas do diretório atual.
- `mkdir`: Cria uma nova pasta.
- `type nul > arquivo.txt`: Cria um arquivo vazio no Windows.
- `cls`: Limpa a tela do terminal.

O que eu fiz: Nesta atividade, eu utilizei apenas o terminal para navegar pelas pastas e criar arquivos, sem usar o mouse.
